# 🌾 SAMARTH: Smart Agriculture Insights Dashboard

**Samarth** is a mini project that analyzes and visualizes agricultural crop production and rainfall data using **Python, SQLite, and Streamlit**.

## 🚀 Features
- Load and manage agricultural datasets (CSV → SQLite)
- Join rainfall and crop production data
- Visualize top crops, rainfall trends, and comparisons
- Interactive chatbot-style dashboard with Streamlit

## 🧩 Tech Stack
- **Python 3**
- **SQLite3**
- **Pandas**
- **Matplotlib**
- **Streamlit**
